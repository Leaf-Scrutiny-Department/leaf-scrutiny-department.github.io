We have used a deep convolutional network which was pre-trained on ImageNet data and finetuned on our dataset. Our model comprises of convolutional layers which perform mathematical operations on the pixel space of the image while retaining its features. These features are processed and stored during the training phase. Our model learns which set of features are closely related to which class of images. Once the training is done we can use this model to predict classes for images which it has never seen before.
