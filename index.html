We have used a deep convolutional network which was pre-trained on ImageNet data and finetuned on our dataset. Our model comprises of convolutional layers which perform mathematical operations on the pixel space of the image while retaining its features. These features are processed and stored during the training phase. Our model learns which set of features are closely related to which class of images. Once the training is done we can use this model to predict classes for images which it has never seen before. The details about model architecture and what features our model captures are given in the following slides.

We have provided intermediate layer outputs and heatmaps to further understand our model performance.Filter outputs help us understand which features of our original image are passed on by our model to subsequent layers. Heatmaps provide us information as to which part of our image was most influential in deciding which class it belongs to.
